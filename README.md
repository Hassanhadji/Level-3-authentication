This project is a secure authentication system built with Node.js, Express, and PostgreSQL.
It demonstrates industry-standard password security practices including bcrypt hashing, automatic salting, and secure credential verification.

The goal of this project was to showcase backend development skills focused on authentication logic and security fundamentals.

Features:

User registration

Secure password hashing using bcrypt

Automatic password salting

Login verification using hash comparison

Duplicate user prevention

PostgreSQL database integration

Environment-based configuration using .env


Tech Stack:

Node.js

Express.js

PostgreSQL

bcrypt

dotenv

EJS (templating)

Security Implementation:

Password Hashing & Salting

Passwords are never stored in plain text.
